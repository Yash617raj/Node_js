
    HTTP status code:- It indicate whether a specific HTTP request has been successfull completed:
    These are group into five response:
    1) Infromational Response(100-199)
    2) Successfull Response(200-299)
    3) Redirectional Response(300-399)
    4) Client error response(400-499)
    5) Server error response(500-599)


    404- Not found
    200- OK(successfull)
    201 - Created
    400 - bad request
    401 - unauthorized
    403 - forbidden (unlike 401 in this the client is known to the server)
    500 - internal server error
    501 - not implemented
